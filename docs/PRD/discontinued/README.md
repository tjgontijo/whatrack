# Archived PRDs

Esta pasta guarda documentos antigos de planejamento, investigações e resumos que deixaram de ser a fonte ativa de decisão.

Critério de arquivamento:

- o documento não reflete mais o produto implementado hoje
- o documento foi superado por mudança de estratégia
- o documento é histórico e não deve guiar a força-tarefa atual de launch

Os PRDs ativos da V1 ficam na raiz de `docs/PRD`.
